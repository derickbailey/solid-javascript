# SOLID JavaScript

A journey through the SOLID software development principles, in JavaScript.

## Run The Code

Install [NodeJS](http://nodejs.org) and run this:

```
npm install
npm start
```

Then go to [http://localhost:3000](http://localhost:3000) in your browser.
